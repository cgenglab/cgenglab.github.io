Welcome to Our Lab

The Interactive Graphics and Engineering Laboartory is in the Creative Informatics Department of the University of Tokyo and is lead by Nobuyuki Umetani. We conduct interdiciplinary research about computer graphics, physics simulation, biomechanics, machine learning, vision and computational fabrication techniques.

Constant Time Median Filter using 2D Wavelet Matrix
In this paper, we have implemented a completely new constant-time median filter based on Wavelet Matrix, which can be efficiently executed by a GPU and supports HDR images.
Yuji Moroto, Nobuyuki Umetani
Paper PDF GitHub Code YouTube (18min)
Fast Editing of Singularities in Field-Aligned Stripe Patterns
In this paper, we propose an algorithm to allow users to interactively edit the singularity positions of field-aligned stripe patterns.
Yuta Noma, Nobuyuki Umetani, Yoshihiro Kawahara

Recolorable Posterization of Volumetric Radiance Fields - Supplemental Video
We investigate artistic posterization of volumetric radiance fields.
Kenji Tojo, Nobuyuki Umetani
Project Page
EnergyHair: Sketch-Based Interactive Guide Hair Design Using Physics-Inspired Energy
This study introduces a new sketch-based hairstyle authoring interface for virtual characters captured using multi-view stereo. The use of physics-related shape optimization in the interface allows natural-looking 3D hair shapes to be modeled from minimal user specifications.
Yuanwei Zhang, Kinuwaki Shinichi, Nobuyuki Umetani

Neural Motion Compression with Frequency-adaptive Fourier Feature Network
We present a neural-network-based compression method to alleviate the storage cost of motion capture data. We leverage this periodicity by applying Fourier features to a multilayered perceptron network.
Kenji Tojo, Yifei Chen, Nobuyuki Umetani
